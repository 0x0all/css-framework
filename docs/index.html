Макет страницы

Макет(layout) сайта создается с помощью портированного и оптимизированного кода из 960 Grid System. 12 колонок, основной блок шириной 60px, расстояние между колонками и рядами 20px.

Основа

Основой разметки служит родительский блок с классом class="g". В нём следует расположить контейнер с классом class="g-row", который разделяет разметку на ряды.

Контейнеры с классами class="g-x" позволяют создавать различные конфигурации колонок в рядах.

<div class="g">
	<div class="g-row">
		<div class="g-5">
			...
		</div><!-- g-5 -->
		<div class="g-7">
			...
		</div><!-- g-7 -->
	</div><!-- g-row -->
</div><!-- g -->

Навигация

Горизонтальная навигация

Для создания простейшей горизонтальной навигации нужно списку задать класс <ul class="f-nav">...</ul>:

<ul class="f-nav">
	<li><a href="#">Первый пункт</a></li>
	<li class="active"><a href="#">Второй активный</a></li>
	<li><a href="#">Третий</a></li>
	<li><a href="#">Последний пункт</a></li>
</ul><!-- f-nav -->

Активный пункт навигации выделяется с помощью класса active.

Получится вот такое навигационное меню:

  • Первый пункт
  • Второй активный
  • Третий
  • Последний пункт

Если предыдущий пример дополнить вложенными списками, получится многоуровневая выпадающая навигация:

<ul class="f-nav">
	<li><a href="#">Выпадающее меню</a>
		<ul>
			<li><a href="#"><strong>Второй уровень</strong></a>
				<ul>
					<li><a href="#"><strong>Третий уровень</strong></a>
						<ul>
							<li><a href="#">...</a></li>
						</ul>
					</li>
				</ul>
			</li>
		</ul>
	</li>
</ul><!-- f-nav -->

«Хлебные крошки»

Навигация типа - «хлебные крошки» создаётся из обычного списка с присвоенным классом <ul class="f-breadcrumbs">...</ul>:

Постраничные ссылки

Класс <div class="f-pager">...</div> служит основой для постраничной навигации. Внутри распологается список ссылок.

Ссылки «Вперёд» и «Назад» отделяются от основных классами .f-pager-prev и .f-pager-next. Активный элемент - классом .active.

<div class="f-pager">
	<ul>
		<li class="f-pager-prev"><a href="#">&#10094; Назад</a></li>
		<li><a href="#">1</a></li>
		<li class="active"><strong>2</strong></li>
		<li><a href="#">3</a></li>
		<li class="f-pager-next"><a href="#">Вперёд &#10095;</a></li>
	</ul>
</div><!-- f-pager -->

Типографика

Заголовки и параграфы

Для всех значимых элементов контента назначен нижний отступ в 14px.

Заголовок первого уровня (H1)

Силовое поле, в согласии с традиционными представлениями, возбуждает циркулирующий гамма-квант - все дальнейшее далеко выходит за рамки текущего исследования и не будет здесь рассматриваться.

Темная материя мономолекулярно испускает кварк, генерируя периодические импульсы синхротронного излучения.

Заголовок второго уровня (H2)

Под воздействием переменного напряжения молекула едва ли квантуема.

Заголовок третьего уровня (H3)

Сверхпроводник бифокально ускоряет барионный экситон по мере распространения сигнала в среде с инверсной населенностью.

Заголовок четвертого уровня (H4)

Неустойчивость, как известно, быстро разивается, если экситон возбуждает лептон, однозначно свидетельствуя о неустойчивости процесса в целом.

Сообщения

Блок с классом .f-message предназначен для вывода сообщений.

Самый простой пример сообщения:

<div class="f-message">
	<span class="close"></span>
	<strong>Внимание</strong>, текст простого сообщения
</div><!--f-message -->
Внимание, текст простого сообщения

Сообщение с заголовком, текстом и кнопками, используется дополнительный элемент .f-message-actions:

<div class="f-message">

	<h5>Заголовок сообщения</h5>
	<p>Пример сообщения с заголовком и кнопками</p>
	<p class="f-message-actions">

	</p>
</div><!--f-message -->

Радио-кнопки и флажки

Простейшая конструкция из минимального набора наобходимых элементов, таких как <label> и <input>.
